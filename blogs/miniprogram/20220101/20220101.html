    小程序开发之深色模式

    Joe Lang 1/1/2022 小程序开发

    如何设置小程序的深色模式主题？首先分为两种情况。

    • 如果手机系统设置了深色模式，小程序可以自动跟随系统的深色模式；
    • 不管手机是否设置了深色模式，在小程序里可以手动切换浅色/深色模式；

    # 跟随系统

    # 配置 app.json

    在app.json中：

    • 配置darkmode为true，即表示当前小程序已适配 DarkMode；
    • 配置themeLocation，定变量配置文件theme.json路径，一般将theme.json文件放在根目录下；
    // app.json
{
    "darkmode": true,
    "themeLocation": "theme.json"
}

    # 创建 theme.json 文件

    在小程序根目录下创建 theme.json 文件，用于颜色主题相关的变量定义，light和dark分别表示浅色和深色模式，以下是我设置的样式变量，仅供参考：

    // theme.json
{
  "light": {
    "bgColor": "#eeeeee",
    "navBgColor": "#eeeeee",
    "navTxtStyle": "black",
    "frontColor":"#000000",
    "tabSelectedColor": "#333333",
    "tabBgColor": "#ffffff",
    "tabFontColor": "#bfbfbf",
    "tabBorderStyle": "white",
    "iconPath1": "images/tabbar/read.png",
    "selectedIconPath1": "images/tabbar/read@select-light.png",
    "iconPath2": "images/tabbar/my.png",
    "selectedIconPath2": "images/tabbar/my@select-light.png"
  },
  "dark": {
    "bgColor": "#333333",
    "navBgColor": "#333333",
    "navTxtStyle": "white",
    "frontColor":"#ffffff",
    "tabSelectedColor": "#ffffff",
    "tabBgColor": "#333333",
    "tabFontColor": "#bfbfbf",

    # 获取当前的主题

    使用wx.getSystemInfo或wx.getSystemInfoSync可以获取当前的主题。

    返回结果中会包含theme属性，值为light或dark。

    # 通过媒体查询进行模式的适配

    如果想在不同的模式下采用不同的样式，可以采用媒体查询的方式。比如：

    /* 浅色模式的样式 */
.some-background {
    background: white;
}
.some-text {
    color: black;
}
/* 浅色模式的样式 */

@media (prefers-color-scheme: dark) {
    /* 深色模式的样式 */
    .some-background {
        background: #1b1b1b;
    }
    .some-text {
        color: #ffffff;
    }
    /* 深色模式的样式 */
}

    跟随系统的深色模式很好设置，也不会有什么问题。难点在于在小程序中如何手动切换模式。

    # 手动切换主题

    要考虑两个问题：

    • 手动切换主题，需要改的样式包括导航栏、页面背景色、tabBar、文字颜色、各种组件的颜色等等；
    • 切换主题后，需要将当前的主题存入缓存，方便在下次进入小程序时保留之前的设置；

    可能每个人实现的思路不太一样，先来说说我的思路。

    # 创建 config.js 文件

    之前已经创建了theme.json，但是由于json文件不能以模块的形式导入导出，所以这里不得不写了冗余的代码。

    通常在项目中都会创建一个config.js文件，用于存放一些全局要用到的配置。如果你还没有创建，那么可以在根目录下创建一个：

    // config.js
export const theme = {
    // 这里是就 theme.json 中的内容
    light: {
        bgColor: "#eeeeee",
        navBgColor: "#eeeeee",
        navTxtStyle: "black",
        frontColor: "#000000",
        tabSelectedColor: "#333333",
        tabBgColor: "#ffffff",
        tabFontColor: "#bfbfbf",
        tabBorderStyle: "white",
        iconPath1: "images/tabbar/read.png",
        selectedIconPath1: "images/tabbar/read@select-light.png",
        iconPath2: "images/tabbar/my.png",
        selectedIconPath2: "images/tabbar/my@select-light.png",
    },
    dark: {
        bgColor: "#333333",
        navBgColor: "#333333",
        navTxtStyle: "white",
        frontColor: "#ffffff",
        tabSelectedColor: "#ffffff",
        tabBgColor: "#333333",
        tabFontColor: "#bfbfbf",
        tabBorderStyle: "black",
        iconPath1: "images/tabbar/read.png",
        selectedIconPath1: "images/tabbar/read@select-dark.png",
        iconPath2: "images/tabbar/my.png",
        selectedIconPath2: "images/tabbar/my@select-dark.png",

    # app.js 中定义 applyTheme()

    applyTheme函数用于设置主题。

    applyTheme()有两种使用场景：

    • 在某页面初始化时，onLoad钩子中调用applyTheme(this)，进行页面初始化；
    • 在手动切换主题时，调用applyTheme(this, 'dark')或applyTheme(this, 'light')；
    // app.js
import { theme } from "./config"

App({
    applyTheme(page, themeName = null) {
        // 页面初始化，从缓存或系统设置中获取主题
        if (themeName === null) {
            themeName = wx.getStorageSync("theme")
                ? wx.getStorageSync("theme")
                : wx.getSystemInfoSync().theme
            if (!wx.getStorageSync("theme"))
                wx.setStorageSync("theme", themeName)
        }
        const curTheme = theme[themeName]
        // 将主题相关的变量注入页面的 data 中
        page.setData({
            bgColor: curTheme.bgColor,
            curTheme: themeName,
        })
        // 手动更改导航栏的样式
        wx.setNavigationBarColor({
            frontColor: curTheme.frontColor,
            backgroundColor: curTheme.navBgColor,
        })
        // 手动更改页面背景色
        wx.setBackgroundColor({
            backgroundColor: curTheme.bgColor,
        })
        // 手动更改 tabBar 的样式
        const tabRoutes = ["home", "my"] // 只有这两个页面有 tabBar
        if (tabRoutes.indexOf(page.route.split("/")[2]) !== -1) {
            wx.setTabBarItem({
                index: 0,
                text: "阅读",
                iconPath: curTheme.iconPath1,
                selectedIconPath: curTheme.selectedIconPath1,
            })
            wx.setTabBarItem({
                index: 1,
                text: "我的",
                iconPath: curTheme.iconPath2,
                selectedIconPath: curTheme.selectedIconPath2,
            })
            wx.setTabBarStyle({
                color: curTheme.tabFontColor,
                backgroundColor: curTheme.tabBgColor,
                selectedColor: curTheme.tabSelectedColor,
            })
        }
    },
})

    准备工作做完了，下面来看看怎么手动切换主题。

    # 调用 applyTheme()

    在每个页面的onLoad钩子中都需要调用applyTheme()来进行页面初始化。

    比如在home页面中：

    // home.js
const app = getApp()

Page({
    data: {
        bgColor: "",
        curTheme: "",
    },
    onLoad: function(options) {
        app.applyTheme(this) // 用于获取 bgColor 和 curTheme
    },
    // 用于切换主题的方法
    onSwitchTheme(e) {
        let curTheme = e.detail.value ? "light" : "dark"
        this.setData({
            curTheme,
        })
        app.applyTheme(this, curTheme)
        wx.setStorageSync("theme", curTheme)
    },
})
